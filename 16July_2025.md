To solve the **"Count Numbers with Exactly 9 Divisors"** problem efficiently, we use a **number theory approach**.

---

### âœ… **Key Observation**

A number `N` has exactly **9 divisors** if:

1. `N = pâ¸` â†’ (where `p` is a prime): divisors = 8 + 1 = 9
2. `N = pÂ² * qÂ²` â†’ (where `p` and `q` are distinct primes):
   Total divisors = (2+1) \* (2+1) = 9

1. **Sieve of Eratosthenes** to precompute all primes up to âˆšn.
2. For `N = pâ¸`: try primes such that `pâ¸ â‰¤ n`
3. For `N = pÂ² * qÂ²`: try all pairs of distinct primes `(p, q)` where `pÂ² * qÂ² â‰¤ n`

* **Time:** `O(âˆšn * log log âˆšn)` for sieve, and \~`O(MÂ²)` for checking pairs (`M` = number of primes â‰¤ âˆšn)
* **Space:** `O(âˆšn)` for primes array

### âœ… Java Code

```java
import java.util.*;

class Solution {
    public static int countNumbers(int n) {
        int limit = (int) Math.sqrt(n) + 1;
        boolean[] isPrime = new boolean[limit];
        Arrays.fill(isPrime, true);
        isPrime[0] = isPrime[1] = false;

        // Sieve of Eratosthenes
        for (int i = 2; i * i < limit; i++) {
            if (isPrime[i]) {
                for (int j = i * i; j < limit; j += i) {
                    isPrime[j] = false;
                }
            }
        }

        ArrayList<Integer> primes = new ArrayList<>();
        for (int i = 2; i < limit; i++) {
            if (isPrime[i]) primes.add(i);
        }

        int count = 0;

        // Case 1: p^8
        for (int p : primes) {
            long num = 1;
            for (int i = 0; i < 8; i++) num *= p;
            if (num <= n) count++;
            else break;
        }

        // Case 2: p^2 * q^2
        int size = primes.size();
        for (int i = 0; i < size; i++) {
            long p2 = 1L * primes.get(i) * primes.get(i);
            if (p2 > n) break;

            for (int j = i + 1; j < size; j++) {
                long q2 = 1L * primes.get(j) * primes.get(j);
                if (p2 * q2 <= n) count++;
                else break;
            }
        }

        return count;
    }
}
```
