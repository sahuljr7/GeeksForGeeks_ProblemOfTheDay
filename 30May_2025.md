**Key Insight**:

  * If node's value > `k`, go **left** to find smaller ones.

### âœ… Java Code (Optimized):

```java
class Node {
    int data;
    Node left, right;

    Node(int value) {
        data = value;
        left = right = null;
    }
}

class Solution {
    public int findMaxFork(Node root, int k) {
        int res = -1;
        while (root != null) {
            if (root.data <= k) {
                res = root.data;
                root = root.right;
            } else {
                root = root.left;
            }
        }
        return res;
    }
}
```

This version uses a **constant space**, **efficient traversal** pattern, and **no recursion**.
